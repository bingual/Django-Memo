[ Linux ]

pwd
	: 현재 작업중인 디렉토리 정보 출력

ls 
	: 현재 경로의 디렉토리 목록을 확인
	: -a 옵션 사용시 모든 파일 검색
	
mkdir <폴더명>
	: 현재 경로에 폴더생성
	: -p 옵션 사용시 하위 디렉터리까지 생성
	
cd <경로주소>
	: 해당하는 경로로 이동
	: /mnt/c/
	
cp
	: 파일 혹은 디렉토리를 복사
	: 디렉토리를 복사할때는 -r 옵션을 주어야함
	
mv
	: 파일 혹은 디렉토리 이동
	: 파일명 변경도 가능
	
rm 
	:파일이나 디렉토리를 삭제
	:디렉토리를 삭제할때는 r 옵션을 주어야 한다.
	:-f 옵션을 주면 사용자에게 삭제 여부를 묻지 않고 바로 삭제한다.
	
cat 
	: 파일내용 출력
	: 파일 합치기
	: 파일 덧붙이기

--------------------------------------------

[ Window CMD ]

dir
	: 현재 경로의 디렉토리 목록을 확인
	: /a 옵션 사용시 모든 파일 검색
	
mkdir <폴더명>
	: 현재 경로에 폴더생성

cd <경로주소>
	: 해당하는 경로로 이동
	
del <파일명>
	: 파일삭제

rd /s <디렉토리명>
	: 디렉토리와 하위에 있는 파일까지 삭제
